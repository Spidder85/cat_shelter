Приют для кошек

Чтобы взять кошку из приюта, достаточно сделать всего несколько шагов:

  1. Изучите таблицу со списком животных, посоветуйтесь с близкими и выберите кошку, которая подойдет именно вам
  2. Отправьте заявку, позвонив нам по номеру телефона
  3. Познакомьтесь с питомцем в приюте и пройдите собеседование с сотрудником приюта, а затем подпишите договор
  4. Питомец ваш!
